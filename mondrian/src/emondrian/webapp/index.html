emondrian

emondrian is a distribution of the emondrian (R)OLAP engine version. The main feature of this version is support of ADOMD.NET and Excel clients.

This package contains:

  • emondrian Binaries
  • Configuration files to deploy an emondrian to a servlet container (like Tomcat, Jetty, etc.)
  • Sample databases

Schema Editor

Use Schema Editor to edit emondrian schema file.

Connect Microsoft Excel to emondrian Cube:

On Excel DATA tab select From Other Sources item and then select From Analysis Services. Enter

http://<your_server_address>:8080/emondrian/xmla

as Server name.
For example,

http://localhost:8080/emondrian/xmla

Connect ADOMD.Net application to emondrian Cube

To connect to emondrian from ADOMD.Net application you must use folowing connection string:

Data Source=http://<your_server_address>:8080/emondrian/xmla;

For example:

Data Source=http://localhost:8080/emondrian/xmla;
